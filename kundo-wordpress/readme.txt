== Description ==

Kundo Wordpress plugin list the latest information from Swedish customersupportsite kundo.se on a dashboard widget. Users can list the latest "FrŒgor", "Problem", "Fšrslag" and "Beršm".

== Installation ==

1. Upload this directory to your /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your account settings under Settings -> Kundo
4. Done
